# Short URL

## Overview

Short URL is a web application that allows users to generate shortened URLs and redirect them to the original links.  
It also tracks visit history for each shortened URL.

The project is built to practice backend development concepts using Node.js, Express, MongoDB, and server-side rendering.

---

## Features

- URL shortening
- Redirection to original URLs
- Visit history tracking
- User authentication
- Role-based access control
- Server-side rendering with EJS

---

## Tech Stack

- Node.js
- Express.js
- MongoDB (Mongoose)
- EJS
- Cookie-based Authentication
